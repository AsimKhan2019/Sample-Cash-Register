Documentation is written with use of markdown syntax.

To find out more about markdown format please visit http://daringfireball.net/projects/markdown/

Files with extension ".mkdn" can be viewed and eddited as simple text files using any text editor.
Within Visual Studio they can be opened with source code editor.

"Markdown Mode" extension provides full taste of markdown syntax for Visual Studio.
It is hosted at http://visualstudiogallery.msdn.microsoft.com/0855e23e-4c4c-4c82-8b39-24ab5c5a7f79/
Also it can be found under "Tools" folder in the repository.
